NPC Creation Tools

Two tools to help you build NPCs from scratch. Use the Stat Tracker to plan your build and spend BP, then use the Stat Block Generator to produce a formatted, downloadable stat block.

📋
NPC Stat Tracker

Track stats and BP spending as you build your NPC from scratch. Set Theme and CL to auto-fill base values, allocate attributes and skills, and download a summary to reference in the generator.

On this site ⚔
NPC Stat Block Generator

Plug in your NPC's details to generate a formatted stat block. Supports attacks, abilities, spells, reactions, and Mythic traits. Save your work as JSON and reload it for future edits.

On this site
Veilstrider Stat Block Generator

Create custom NPC Veilstrider stat blocks for ship combat encounters in the Veil.

Leviathan Stat Block Generator

Build stat blocks for massive Leviathan-class creatures encountered across the Veil.
