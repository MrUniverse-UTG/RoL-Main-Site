NPC Stat Block Generator

Notes about using the Stat Block Generator

  • If you check Mythic, do not adjust HP/DP/MP manually — they will be auto-scaled when you generate.
  • The Mythic checkbox will also automatically add all Mythic traits to the stat block.
  • Opposing modifiers for skills are calculated automatically.
  • Adding an Ability Standard Action will auto-add the Charging quick action.
  • The Save JSON button saves your work; Load JSON resumes editing later.
  • The JSON saved in the Stat Tracker is separate from the JSON saved in this generator. They are not compatible with each other.
